SkyConnect is an innovative in-flight networking platform designed for professionals to connect, engage, and expand their professional network while flying. This project was developed during a hackathon from November 2-5, 2023, with American Airlines as the headline sponsor.

Project Name
SkyConnect

Challenge Statement(s) Addressed ðŸŽ¯
SkyConnect addresses the challenge of unproductive downtime during flights and the missed opportunity for professional networking. In an era where every minute counts, SkyConnect transforms travel time into a networking hub, fostering connections and collaborations amongst professionals.

Project Description ðŸ¤¯
SkyConnect leverages flight time to enable passengers to discover professional opportunities, connect with fellow travelers, and potentially create new business ventures. By integrating LinkedIn profiles, the platform matches professionals based on their industry, interests, and travel itineraries. The real-time chat feature facilitates immediate communication, making every flight a networking event.

Project Value ðŸ’°
SkyConnect targets business professionals who value networking and productive time usage. By providing a platform for in-flight professional connections, SkyConnect delivers tangible benefits like increased business opportunities, enhanced travel experiences, and optimized networking time. It's an exclusive service that adds value to the airline's offerings, potentially increasing customer loyalty and satisfaction.

Frontend Development (My Contribution)
I spearheaded the frontend development of SkyConnect, using React.js to create a dynamic and responsive user interface. My primary focus was on building key components such as:

  • Login Page: Integrated with LinkedIn OAuth for a seamless login experience.
  • Dashboard: Displayed suggested connections and travel updates in an intuitive layout.
  • Profile Page: Allowed users to view and edit their professional information, synced with LinkedIn.
  • Chat Component: Enabled real-time messaging between users, enhancing the networking experience.

I also worked closely on the UI/UX design, ensuring the app was user-friendly and visually appealing. Using Bootstrap, I designed the layout to be responsive and accessible across various devices, making it convenient for users to interact with the app during flights.

Backend Development
Santosh Lamichhane and Buchizya Mwase managed the backend services, including user authentication and database management. They implemented:

  • User Authentication: Using LinkedIn OAuth for secure login.
  • Database Management: Utilizing Firebase Realtime Database to store user profiles, chat messages, and travel statuses.
  • Matchmaking Algorithm: Developed an algorithm to match users based on their professional interests and travel details.

API Integrations
Tanyaradzwa Chisepo integrated essential third-party APIs, including:

  • American Airlines API: For retrieving flight information and user-related data.
  • Google Maps API: For airport navigation and wayfinding.
  • OpenWeatherMap API: Provided weather forecasts for users' travel destinations.
