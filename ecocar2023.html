Customizable Facial Gesture Recognition System

Project Overview

This project focuses on developing an advanced facial gesture recognition system aimed at enhancing driving assistive technology. The system utilizes a combination of cutting-edge algorithms and technologies to monitor and analyze driver behavior, improving overall road safety.

Technologies and Algorithms

  • MediaPipe: Employed for real-time image processing, MediaPipe provides a robust framework for detecting and tracking facial landmarks, facilitating accurate and efficient gesture recognition.
  • OpenCV: Utilized for facial recognition, OpenCV helps in detecting and analyzing facial features to assess the driver's engagement level.
  • Eye Aspect Ratio (EAR): Used to determine whether the driver is awake or not. By analyzing the ratio of the eyeâ€™s vertical and horizontal distances, the system can identify drowsiness or sleep.
  • Mouth Aspect Ratio (MAR): Implemented to recognize yawns or signs of fatigue. However, the current implementation struggles with distinguishing between thick lips and an open mouth.
  • dlib: Leveraged for facial landmark detection, dlib provides precise facial feature localization to support gesture and expression analysis.

Project Objectives

  • Detect if the user is actively engaged or awake.
  • Identify when the user is distracted by tracking gaze direction.
  • Determine when the user has fallen asleep or closed their eyes.
  • Recognize yawns or signs of fatigue.
  • Develop and implement a driving alert system using facial expression and gesture recognition to send alerts to the driver.

Current Status and Future Work

Currently, the system can successfully detect whether the user is awake or drowsy. However, the mouth detection feature has limitations, particularly in distinguishing between thick lips and an open mouth. Future work will focus on refining mouth detection capabilities to enhance accuracy. Additionally, the system will be integrated into the ECOCAR by developing a comprehensive driver monitoring system to further improve driving safety and assistive technology.
